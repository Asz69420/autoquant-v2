Open Claw just dropped V2026.3.7 and this one has GPT 5.4 support, Gemini 3.1 flashlight, a fix for the ACP binding bug that was breaking every restart, and a slim Docker build that finally stops the image from being absurdly oversized. Version 2026.3.7 shipped within the last 24 hours, and it is a focused release. No massive breaking changes, no 150 fix change log. Four targeted upgrades that directly address the most common pain points users have been hitting since 2026.3.2. Let's go through each one. First, GBT 5.4 and Gemini 3.1 Flashlight both get official ACP bindings in this release. If you are manually configuring these models through workarounds or community forks, that is now unnecessary. Both models are first class citizens in the ACP layer with full official support. Second, the ACP binding persistence fix. This one was genuinely painful. Every time you restarted Open Claw, your ACP bindings reset, meaning any custom model routing, any specialized agent configuration tied to an ACP binding had to be manually reconfigured from scratch after every restart. That is now fixed. Bindings persist across restarts. exactly as you'd expect them to. Third, slim Docker multi-stage build. The Open Claw Docker image has been notoriously large, a problem for anyone deploying in containerized environments with image size constraints or slow pool times. The new multi-stage build strips everything that isn't needed at runtime, bringing the image size down significantly. Fourth, Secret Ref coverage now extends to gateway authentication. Previously, Secret Ref handled credentials for connected services, but gateway off was a separate manual configuration. Now, it's unified. One credential management system for everything OpenClaw touches. No breaking changes, four targeted fixes, two major model additions. This is the kind of polish release that makes production deployments actually stable. Update immediately.
