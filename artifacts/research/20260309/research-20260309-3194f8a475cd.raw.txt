Open Claw just dropped version 2026.3.2 on March 3rd and buried inside 150 fixes is a feature that fundamentally changes how AI agents handle documents. This release ships a native PDF analysis tool backed by Anthropic and Google as processing backends, a fully matured secret system supporting 64 credential targets, and critical security patches across the gateway and plug-in layers. Previously, getting Open Claw to read a PDF meant writing a custom skill to extract text manually and pipe it into the LLM context. Now the agent natively ingests PDFs endto end. Feed it a 200page technical spec and it reasons over the full document without any pre-processing. The secret system is equally significant. Secret ref now covers 64 credential targets including Stripe, Slack, and GitHub and unresolved references now fail fast and loudly instead of silently breaking midworkflow. On the security side, this release patches gateway loop back websocket hardening, skill workspace symbolic link escape protection and plug-in root dduplication, closing three separate classes of privilege escalation vulnerabilities in one shot. Native PDF reasoning, plus a bulletproof secret system in one release. This is Open Claw becoming infrastructure.
